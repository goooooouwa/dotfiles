dotfiles - vim, git and zsh customization 
========

# Prerequisites 

Install [Janus](https://github.com/carlhuda/janus)
Install [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)


# Installation
`$ make install`

# customization list

## Vim
* colorscheme molokai
* tab navigation key mapping

## Git
* command shortcuts
* beautiful log
* colorful info
